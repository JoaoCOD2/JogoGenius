Como Jogar

Objetivo: memorize a sequência de luzes e sons e repita cada rodada corretamente. A cada fase, o padrão aumenta e o jogo fica mais rápido.

Como ganhar pontos: cada rodada correta soma pontos extras e aumenta seu recorde. Quanto mais fases você chegar, maior será sua pontuação.

  1. Clique em "INICIAR" para começar a partida.
  2. Observe a sequência exibida no modo escolhido.
  3. Repita a sequência clicando nos botões coloridos na mesma ordem.
  4. A cada rodada, uma nova cor é adicionada à sequência.
  5. Atalhos de teclado:
    • G para Verde
    • R para Vermelho
    • Y para Amarelo
    • B para Azul
    • Espaço para iniciar
  6. Quanto mais fases você avança, mais rápido fica o jogo.
  7. Errando uma cor, o jogo termina.
  8. Acumule pontos e tente bater recordes!

Modos de dificuldade

  • Fácil — a sequência fica visível no painel o tempo todo. Ideal para iniciantes.
  • Médio — você pode visualizar a sequência completa apenas 5 vezes por partida. Use as visualizações com cuidado.
  • Difícil — não há ajuda visual da sequência. Você deve memorizar apenas pelos flashes dos botões.

Sobre o Jogo

GENIUS é um clássico jogo de memória que desafia sua capacidade de repetir sequências cada vez mais longas.

O jogo foi originalmente criado pela Milton Bradley em 1978 com o nome "Simon". Essa versão foi desenvolvida com tecnologias web modernas:

  • HTML5 — Estrutura semântica
  • CSS3 — Design responsivo e animações
  • JavaScript — Lógica de jogo e interatividade
  • Web Audio API — Sons sintetizados em tempo real
  • LocalStorage — Persistência de recordes e rankings

Teste seus limites de memória e torne-se um mestre do GENIUS!
