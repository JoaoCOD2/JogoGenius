# 🧠 GENIUS — Jogo de Memória

Um jogo de memória inspirado no clássico **Simon**, desenvolvido com **HTML, CSS e JavaScript** como projeto escolar.

---

## 📖 Sobre o Jogo

O objetivo do jogo é repetir a sequência de cores mostrada na tela.  
A cada rodada, uma nova cor é adicionada, deixando o desafio mais difícil.

---

## 🎮 Como Jogar

1. Clique em **Iniciar**
2. Observe a sequência de cores
3. Repita clicando nos botões na mesma ordem
4. Acerte para avançar de fase
5. Se errar, o jogo termina

---

## ✨ Funcionalidades

- Sistema de fases
- Aumento de dificuldade
- Sons e efeitos visuais
- Ranking com recorde salvo
- Navbar com informações do jogo
- Interface responsiva
